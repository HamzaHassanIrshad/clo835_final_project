This is a CLO835 Final Project demonstrating:

  • Containerized Flask application
  • MySQL database integration
  • Background image from S3 bucket
  • Kubernetes deployment
  • GitHub Actions CI/CD
  • Amazon EKS cluster
  • Persistent Volume Claims (PVC)
  • LoadBalancer service
  • Horizontal Pod Autoscaler (HPA)
  • IAM Roles for Service Accounts (IRSA)
  • ConfigMap for configuration
  • Secrets for database credentials
  • Amazon ECR container registry

This application allows you to add and retrieve employee information from a MySQL database with persistent storage and auto-scaling capabilities.

Key Features:

  • Dynamic background images from S3
  • Database persistence across pod restarts
  • Automatic scaling based on CPU usage
  • Secure credential management
  • Internet-accessible via LoadBalancer
